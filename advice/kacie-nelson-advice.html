HTML tips to remeber!

Always close your HTML tags

This will ensure that you HTML will work properly on all browsers. While there are some tags that don't have a corresponding closing tag, most will, so just be aware of which ones don't.

Style HTML using style sheets when possible

Style sheets make the look of your coding much easier to control and edit when needed.

Use an HTML validator

Running your web pages through an HTML validator is a great way to catch mistakes in your coding you might have missed. Your pages may appear fine on one browser and fail to work on another. A validatior will catch this for you.

Use HTML comments wisely

Adding comments to your coding can make it much easier to find specific setions and their functions. This also makes it eaiser for others to understand your coding if you are woking with a team.

Use widths and heights with HTML images

It's a good idea to specify the width and height of an image. This allows the browser to format the page faster as it is loaded.

Don't forget about online resources!

If you are ever stuck with your HTML or CSS, remember there are so many outlets online to help you work your way through.
