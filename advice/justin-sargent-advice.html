Hi, my name is Justin and I have some advice for you!

Learn OOP Principles

Nearly all popular modern programming languages are OOP.
Don't overthink "what programming language should I learn?".
Learn whatever programming language is best for the project you're developing.

After learning OOP principles you'll realize that most popular modern programming languages are very much alike.
It is just a matter of getting used to a programming language's idioms.
